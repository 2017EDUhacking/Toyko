# 「city country city」邊用餐能邊欣賞音樂
```
由日本知名音樂人所開設的「city country city」隱身在老舊公寓裡，結合咖啡廳與二手黑膠唱片行，
既提供美食也提供精神食糧，在用餐區旁邊有小DJ台，店員會不時地更換曲目，一邊用餐一邊享受音樂，是很愉快的體驗呢！
店內的二手唱片存量雖不算多，但可一窺店主人的獨到選品眼光，有許多市面上找不到的珍稀唱盤。
```
